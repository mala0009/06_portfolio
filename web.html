Opgave

I temaet “Grundlæggende web” blev vi introduceret for redskaberne HTML, CSS, Grid, Flex og Media Querires. Størstedelen af temaet blev brugt på at kode det website som udgjorde studiestartsprøven. Ved hjælp af de forskellige redskaber som vi havde lært, skulle vi kode og lave et responsivt website som levede op til kravene. Nedenunder er der linket til min studiestartsprøve.
